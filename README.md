# ERP Marketplace AI

Trendyol, Hepsiburada, Amazon Türkiye, N11, İdefix ve Pazarama'ya entegre olan, AI destekli dinamik fiyatlandırma özelliğine sahip bir ERP sistemi.

## Özellikler

### 📦 Ürün Yönetimi
- Ürün ekleme, düzenleme, silme
- Pazaryeri senkronizasyonu
- Stok takibi
- SKU yönetimi

### 📊 Sipariş Yönetimi
- Tüm pazaryerlerden sipariş takibi
- Sipariş durumu güncellemeleri
- Müşteri bilgileri

### 💰 Pazaryeri Entegrasyonu
- Trendyol
- Hepsiburada
- Amazon Türkiye
- N11
- İdefix
- Pazarama

### 🤖 AI Fiyat Analizi Robotu
- Rakip fiyat izleme
- Dinamik fiyatlandırma
- Kar/zarar analizi

### 📈 Kar/Zarar Hesabı
- Maliyetler (manuel, Excel, fatura girişleri)
- Kargo ücretleri
- Pazaryeri komisyonları

## Kurulum

### Gereksinimler
- Node.js 16+
- PostgreSQL 12+
- Appsmith

### Adımlar

1. Repository'yi klonla
2. Backend kurulumunu yap
3. Veritabanını oluştur
4. Sunucuyu başlat
5. Appsmith'i kurulum
